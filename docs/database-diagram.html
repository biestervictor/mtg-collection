Relations:
  • card-collection.user → import-history.user (tracks imports per user)
  • card-collection.setCode → scryfall-cards.setCode (card lookup)
  • card-collection.setCode → scryfall-sets.setCode (set info)
ImportedCardInfo Structure:
  • name: String
  • setCode: String
  • collectorNumber: String
  • quantity: int
  • foil: boolean
Notes:
  • card-collection: User-spezifische Sammlung. Key = setCode + collectorNumber + foil
  • scryfall-cards: Gecachte Scryfall-Daten. Preise werden nachts aktualisiert, Bilder bleiben unverändert
  • import-history: Import-Versionierung mit Tracken von New/Removed Cards
